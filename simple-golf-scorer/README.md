# Simple Golf Scoring App

A lightweight, browser-based golf scoring application that runs entirely in your browser. No server or database required!

## Features

- Multiple game types (Stroke Play, Skins, Wolf, Scramble)
- Add/remove players
- Score tracking for 18 holes
- Automatic total score calculation
- Navigation between holes
- Clean, responsive design

## How to Use

1. Download the files:
   - `index.html`
   - `app.js`

2. Open `index.html` in your web browser

3. To start a new game:
   - Add player names (minimum 2 players)
   - Select game type
   - Click "Start Game"

4. During the game:
   - Enter scores for each hole
   - Use Previous/Next buttons to navigate between holes
   - View total scores automatically calculated

## Game Types

- **Stroke Play**: Traditional scoring where lowest total score wins
- **Skins**: Each hole is worth a skin, lowest score wins the skin
- **Wolf**: Players take turns being the "wolf" and can choose partners
- **Scramble**: Team format where players choose the best shot

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5.1.3
